Скриншот 7: Определение MX-записей домена kubsu.ru с помощью nslookup

На этом скриншоте я выполнила следующую часть третьего пункта лабораторной работы, запросив MX-записи (записи почтовых серверов) для домена kubsu.ru.

Я ввела команду nslookup -type=mx kubsu.ru.

Система снова отобразила информацию об используемом DNS-сервере.

В секции "Non-authoritative answer" я получила список MX-записей для домена kubsu.ru. Эти записи указывают на почтовые серверы, ответственные за прием электронной почты для данного домена:

  • kubsu.ru mail exchanger = 20 mx.kubannet.ru.
  • kubsu.ru mail exchanger = 10 mx2.kubannet.ru.
  • kubsu.ru mail exchanger = 50 mx.kubsu.ru.

Числа 20, 10, 50 перед именами серверов являются приоритетами: чем меньше число, тем выше приоритет. Таким образом, mx2.kubannet.ru. имеет самый высокий приоритет (10), а mx.kubsu.ru. — самый низкий (50).

Пункт 4: Определение даты регистрации доменов с помощью whois

Скриншот 5: Определение даты регистрации домена kubsu.ru с помощью whois

На этом скриншоте я выполнила часть четвертого пункта лабораторной работы, запросив информацию о регистрации домена kubsu.ru с помощью команды whois.

Я ввела команду whois kubsu.ru.

В ответ сервер предоставил детальную информацию о домене. Среди этих данных:

  • domain: KUBSU.RU - подтверждение запрошенного домена.
  • state: REGISTERED, DELEGATED, VERIFIED - статус домена.
  • org: Kuban State University - организация, которой принадлежит домен.
  • registrar: RU-CENTER-RU - регистратор домена.
  • created: 1998-03-28T12:18:30Z - это и есть искомая дата регистрации домена, 28 марта 1998 года.

Также указаны даты paid-till (до какого числа оплачена регистрация) и free-date (когда домен освободится, если не будет продлен). В конце также указано Last updated on 2026-02-20T17:28:01Z, что является датой последнего обновления записи whois.

Скриншот успешно демонстрирует получение даты регистрации домена kubsu.ru, которая составляет 28 марта 1998 года.
